CSS Property

The property is the color piece of this declaration. It dictates which aspect of the selector will be changed visually. The value is what the chosen CSS property will be changed to.
for example, we use the hex value of #000, to change color property to "black."

There are a number of CSS properties that you can use to style a site. Some examples are :

  • Border (including border-style, border-color, and border-width)
  • Padding (including padding-top, padding-right, padding-bottom, and padding-left)
  • Margins (including margin-top, margin-right, margin-bottom, and margin-left)
  • Font-family
  • Font-size
  • Background-color
  • Height
  • Width
